# Expansion Track Roadmap (Post-MVP)

## Phase E1: Scheduler Foundation
- Add recurring task definitions and run windows
- Add task template + parameterized prompt model
- Add scheduler triggers (manual, cron-like, dependency-driven)

## Phase E2: Agent Pipeline Orchestration
- Represent workflows as DAG/pipeline steps
- Add handoff events and output contracts between agents
- Add retry/fallback policies per step

## Phase E3: Provider Routing
- Add per-agent provider configuration (local/API/OAuth)
- Add provider health checks and runtime selection policy
- Add provider timeout/retry budgets by task type

## Phase E4: Mission Control Expansion
- Add recurring job manager UI
- Add pipeline visualizer and execution timeline
- Add approval queue and escalations

## Phase E5: Mac mini Operationalization
- Add bridge process deployment profile
- Add supervised services and startup orchestration
- Add backup/restore and operational playbooks
